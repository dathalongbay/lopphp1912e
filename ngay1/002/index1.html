HTML gồm các thẻ mở và các thẻ đóng
        thẻ đóng có dấu / trước tên thẻ
        ví dụ :
        body là thẻ mở
        có /body là thẻ đóng
        thẻ tiêu đề h1,h2,h3,h4,h5,h6

        cách để chạy file html là mở trực tiếp file trong ổ đĩa
        và click đúng vào file html đó
