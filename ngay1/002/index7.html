Thẻ label là thẻ nhãn
        thẻ input là thẻ nhập thông tin
        thẻ input có thuộc tính type chứa kiểu
        type="text" có nghĩa là thẻ nhập văn bản
        thẻ input có thuộc tính name
        thẻ input có thuộc tính value chứa giá trị của nó
        thẻ input có thuộc tính placeholder
        thuộc tính placeholder sẽ hiện ra văn bản gợi ý cho người dùng
        nhập thông tin khi value rỗng chưa có giá trị
