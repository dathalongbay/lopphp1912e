Thuộc tính padding trong css
        Thuộc tính padding tạo ra 1 khoảng không gia xung quanh content của thẻ html
        Vậy thuộc tính margin khác padding ?
        1 - Thuộc tính margin tạo ra 1 khoảng không gian xung quanh thẻ html
        2 - margin nằm ngoài border của thẻ html
        còn padding nằm trong border của thẻ html
        
        BOX MODEL ( Mô hình hộp ) trong CSS gồm 4 thành phần :
        1 - margin
        2 - border
        3 - padding
        4 - content
