thuộc tính lặp ảnh nền
        thuộc tính background-repeat
        4 giá trị
        repeat ( giá trị mặc định ) lặp ảnh nền
        no-repeat ( không lặp ảnh nền )
        repeat-x ( lặp ảnh nền theo chiều ngang )
        repeat-y ( lặp ảnh nền theo chiều dọc )

        chu ý :
        thuộc tính background-repeat chỉ có tác dụng
        khi kích cỡ của thẻ html lớn hơn so với ảnh nền
