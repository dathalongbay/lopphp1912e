Khái niệm của CSS
        CSS có tác dụng trang trí cho các thẻ html về màu sắc , kích cỡ,
        cách hiển thị , độ rộng, độ cao ...
        Phân biệt kỹ thuộc tính html và thuộc tính của css
        *** Thuộc tính của html có tác dụng bổ sung đặc điểm tính chất cho thẻ html
        ví dụ thẻ img có thuộc src chứa đường dẫn của hình ảnh
        thẻ a có thuộc href chứa liên kết của thẻ a
        *** Thuộc tính của css sẽ chịu trách nhiệm thay đổi màu sắc , kích cỡ , độ rộng , độ cao ...
        cho thẻ html
        Làm quen 1 số thuộc tính css đầu tiên
        background-color ( màu nền )
        color ( màu chữ )
        width ( chiều rộng và đơn vị là pixel )
        height ( chiều cao và đơn vị là pixed )
        font-size ( kích cỡ của chữ đơn vị là pixel )

        Thêm các thuộc tính css vào 1 thẻ html
        thông qua thuộc tính style của thẻ html ví dụ như sau
        Cú pháp
        style="thuộc tính css số 1: giá trị;thuộc tính css số 2 : giá trị; thuộc tính css số 3: giá trị; ..."
